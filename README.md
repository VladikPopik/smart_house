***
## Чтобы запустить сервис нужно выполнить следующее:
### sudo docker compose up -d

### cd my-react-app
### npm run dev
***
***
Это мой MVP-сервис умного дома, основанный на микросервисной архитектуре на языке Python, с фронтендом на JS/React, с использованием docker, kafka, и на платформе Raspberry PI 4 model B
